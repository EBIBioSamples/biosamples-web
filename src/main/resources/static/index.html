BioSamples - database of sample descriptions

The BioSamples database aggregates sample information for reference samples (e.g. Coriell Cell lines) and samples for which data exist in one of the EBI's assay databases such as ArrayExpress, the European Nucleotide Archive or PRIDE. It provides links to assays for specific samples, and accepts direct submissions of samples information.

Info

  • Help pages about how to search BioSamples, how to submit data, and FAQ.
  • Programmatic access to query and download data using web services.
  • Latest news about BioSamples database.
  • The BioSamples database now offers access to RDF, and a supporting SPARQL endpoint as part of the EBI's RDF platform.
