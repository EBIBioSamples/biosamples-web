BioSamples for Crop Plant Bioinformatics

Examples: Arabidopsis Thaliana, root, salt stress, SAMEA3132246,

The BioSamples database is a repository of sample information. BioSamples contains descriptions and metadata on biological source materials. Samples are collected either because they are 'reference' samples (for example, Coriell cell lines or 1000 genome samples) or because they have been used in experiments that has generated data that exists in an assay database such as the European Nucleotide Archive, ArrayExpress or PRIDE.

This is a dedicated Crop Informatics instance of Biosamples, and it contains metadata from all publicly available crop plant samples, cross linked to assay data from ENA, Ensembl Plants, ArrayExpress, Expression Atlas and more.

For more information on this project or the biosamples database, please get in touch by emailing us at biosamples@ebi.ac.uk
